= blog

* http://github.com/dalibor/blog

== DESCRIPTION:

Personal Blog application for http://dalibornasevic.com

== INSTALL:

- Clone repository

    git clone git://github.com/dalibor/blog.git
    cd blog

- Config database

    cp config/database.yml.template config/database.yml
    vi config/database.yml

- Personalize application

    cp config/application.yml.template config/application.yml
    vi config/application.yml

- Setup google analytics

    cp config/google.yml.template config/google.yml
    vi config/google.yml

- Change the secret token for verifying the integrity of signed cookies

    First generate new secret token with: 'rake secret'
    Then add it to config/initializers/secret_token.rb

- Install gems

    sudo gem install rack # be sure to have the latest rack gem installed
    sudo rake gems:install # install other gems

- Create database and run application

    rake db:create
    rake db:migrate
    ruby script/server

== SEO

- When creating post fill description area for description meta tag of your show post page

- Add tags to the post which are used as keywords on that page

- Run rake sitemap task to create a sitemap and submit it to search engines

    rake sitemap:refresh

== Run tests

- Run cucumber features

    cucumber features

- Run specs

    spec spec --options spec/spec.opts
